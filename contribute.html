The TripleA community is large and growing, and could use you! There are many ways to jump in and help out.

Code

TripleA is a GitHub open source project and is seeking volunteers to help improve the code and add features. There are a number of bug fixes and code refactorings that can be done easily as training projectx. The project wiki has instructions on how to get set up and running with the game engine code.
